## 6. Workflow predykcji

### **Przykładowy workflow operacyjny**
1. system pobiera dane z modułów,  
2. AI analizuje dane historyczne,  
3. model generuje prognozę,  
4. automatyzacja reaguje na prognozę,  
5. system tworzy zadania lub alerty,  
6. manager podejmuje decyzje.

### **Workflow automatyczny**
1. IF prognoza opóźnienia > 60%  
2. THEN utwórz zadanie „analiza harmonogramu”  
3. THEN wyślij alert do managera  
4. THEN aktualizuj Gantt.

---

## 7. Automatyzacje predykcji

Moduł AI Predykcja współpracuje z silnikiem automatyzacji IF→THEN.

### **Przykłady automatyzacji**
- IF prognoza ryzyka > 70% → THEN utwórz zadanie,
- IF prognoza kosztów > budżet → THEN wyślij alert,
- IF prognoza konwersji spada → THEN uruchom kampanię reaktywacyjną,
- IF prognoza zaległości > 50% → THEN wyślij przypomnienie,
- IF prognoza opóźnień > 40% → THEN aktualizuj harmonogram.
