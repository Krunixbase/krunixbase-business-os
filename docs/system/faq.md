# **FAQ — Krunixbase Business OS**

## ❓ Często zadawane pytania

---

## **1. Czym jest Krunixbase Business OS?**
Krunixbase to modularny system operacyjny dla biznesu, który łączy CRM, projekty, budżety, KSeF, płatności, automatyzacje, BI i AI w jednym ekosystemie.  
Pełna lista modułów: **[moduły systemu](moduly_systemu.md)**

---

## **2. Czy Krunixbase jest open‑source?**
Tak. Projekt jest w pełni open‑source i otwarty na współtworzenie.  
Repozytorium: github.com/Krunixbase/krunixbase-business-os

---

## **3. Jak zacząć korzystać z Krunixbase?**
Przejdź do przewodnika: **[Get Started](get_started.md)**  
Znajdziesz tam instalację, konfigurację, API i pierwsze moduły.

## **4. Jakie technologie wykorzystuje Krunixbase?**
- Node.js  
- PostgreSQL  
- REST API  
- Docker / CI/CD  
- JSONB  
- automatyzacje IF→THEN  
- BI + AI predykcyjne  

Pełna architektura: **[moduł architektura](modul_architektura.md)**

## **9. Jak wygląda bezpieczeństwo systemu?**

- TLS 1.3  
- szyfrowanie danych  
- role i uprawnienia  
- audyt działań  
- zgodność z RODO  

Dokumentacja: **[moduł bezpieczeństwo](modul_bezpieczenstwo.md)**
