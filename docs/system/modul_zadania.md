# Moduł Zadania i Workflow — Krunixbase Business OS

Moduł zadań jest odpowiedzialny za zarządzanie pracą operacyjną w firmie. Umożliwia tworzenie zadań, przypisywanie ich do użytkowników, monitorowanie postępu, zarządzanie projektami oraz automatyzację działań. Jest w pełni zintegrowany z CRM, kampaniami, automatyzacjami IF→THEN, raportami oraz modułem KSeF.

---

## 1. Cele modułu zadań

- organizacja pracy zespołu,
- zarządzanie procesami operacyjnymi,
- automatyzacja powtarzalnych działań,
- monitorowanie postępu,
- integracja z innymi modułami systemu,
- zwiększenie efektywności pracy.

## 4. Workflow zadań

### **Przykładowy workflow operacyjny**
1. klient przechodzi do etapu „oferta”,  
2. automatyzacja tworzy zadanie „przygotuj ofertę”,  
3. operator wykonuje zadanie,  
4. status zadania zmienia się na „zakończone”,  
5. automatyzacja wysyła kampanię ofertową.

### **Przykładowy workflow projektowy**
1. utworzenie projektu,  
2. dodanie etapów,  
3. dodanie zadań projektowych,  
4. przypisanie zadań,  
5. monitorowanie postępu,  
6. raport projektu.

---

## 5. Automatyzacje zadań

Moduł zadań współpracuje z silnikiem automatyzacji IF→THEN.

### **Przykłady automatyzacji**
- IF klient przejdzie do etapu „oferta” → THEN utwórz zadanie,
- IF kampania zakończona → THEN utwórz zadanie analityczne,
- IF faktura pobrana → THEN utwórz zadanie księgowe,
- IF zadanie opóźnione → THEN wyślij powiadomienie.
