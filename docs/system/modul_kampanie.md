# Moduł Kampanie — Krunixbase Business OS

Moduł kampanii marketingowych umożliwia tworzenie, planowanie, wysyłanie i analizowanie kampanii komunikacyjnych w wielu kanałach. Jest w pełni zintegrowany z CRM, automatyzacjami IF→THEN, zadaniami oraz raportami.

---

## 1. Cele modułu kampanii

- prowadzenie komunikacji z klientami,
- automatyzacja działań marketingowych,
- zwiększenie konwersji i aktywności klientów,
- integracja kampanii z CRM i workflow,
- pełna analityka działań marketingowych.

---

## 4. Workflow kampanii

### **Przykładowy workflow kampanii e-mail**
1. utworzenie kampanii,  
2. wybór segmentu klientów,  
3. wybór szablonu,  
4. ustawienie harmonogramu,  
5. wysyłka kampanii,  
6. zbieranie statystyk,  
7. automatyzacja IF→THEN reaguje na wyniki kampanii.

### **Przykładowy workflow kampanii automatycznej**
1. IF klient zmieni status → THEN wyślij kampanię powitalną,  
2. kampania wysyła e-mail,  
3. CRM zapisuje aktywność,  
4. automatyzacja tworzy zadanie dla operatora.

---

## 5. Automatyzacje kampanii

Moduł kampanii współpracuje z silnikiem automatyzacji IF→THEN.

### **Przykłady automatyzacji**
- IF klient nieaktywny 30 dni → THEN wyślij kampanię reaktywacyjną,
- IF klient przejdzie do etapu „oferta” → THEN wyślij kampanię ofertową,
- IF kampania zakończona → THEN utwórz raport,
- IF klient kliknie link → THEN przypisz zadanie operatorowi.
