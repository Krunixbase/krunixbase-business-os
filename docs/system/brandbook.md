# 2. Logo





## 2.1 Konstrukcja
Logo składa się z:
- centralnej litery **K**,
- heksagonu symbolizującego modułowość,
- połączeń prowadzących do ikon modułów.

## 2.2 Warianty
- pełne logo (z ikonami modułów),
- logo uproszczone (sam heksagon + K),
- logo monochromatyczne.

## 2.3 Zasady użycia
- minimalny margines: 24px,
- nie zmieniać proporcji,
- nie stosować efektów 3D,
- nie zmieniać kolorów poza paletą systemową.
