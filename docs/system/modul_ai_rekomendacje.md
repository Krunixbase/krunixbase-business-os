## 4. Typy rekomendacji

### **Rekomendacje operacyjne**
- „Skontaktuj się z klientem X”
- „Wyślij kampanię do segmentu Y”
- „Przesuń zadanie Z o 2 dni”
- „Dodaj koszt do budżetu projektu”

### **Rekomendacje strategiczne**
- „Projekt A jest zagrożony — wykonaj analizę”
- „Partner B ma wysoką skuteczność — zwiększ priorytet”
- „Kampania C ma spadek konwersji — uruchom wersję alternatywną”

### **Rekomendacje predykcyjne**
- „Prognoza opóźnień > 60% — dodaj zadanie kontrolne”
- „Prognoza kosztów > budżet — wykonaj analizę finansową”
- „Prognoza ryzyka > 70% — dodaj działania zapobiegawcze”

## 5. Workflow rekomendacji

### **Przykładowy workflow operacyjny**
1. system analizuje dane CRM, projektów, kampanii, budżetów,  
2. AI generuje rekomendacje,  
3. operator zatwierdza lub odrzuca rekomendację,  
4. automatyzacja wykonuje działanie,  
5. BI aktualizuje raporty.

### **Workflow automatyczny**
1. IF rekomendacja krytyczna  
2. THEN utwórz zadanie  
3. THEN wyślij alert  
4. THEN aktualizuj moduły powiązane.

---

## 6. Automatyzacje rekomendacji

Moduł AI Rekomendacje współpracuje z silnikiem automatyzacji IF→THEN.

### **Przykłady automatyzacji**
- IF rekomendacja CRM → THEN utwórz zadanie kontaktowe,
- IF rekomendacja projektowa → THEN aktualizuj harmonogram,
- IF rekomendacja finansowa → THEN aktualizuj budżet,
- IF rekomendacja windykacyjna → THEN wyślij przypomnienie,
- IF rekomendacja partnerska → THEN wygeneruj prowizję.
