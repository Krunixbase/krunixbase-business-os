## 6. Workflow BI

### **Przykładowy workflow analityczny**
1. system pobiera dane z modułów,  
2. BI agreguje dane,  
3. BI generuje dashboardy,  
4. predykcja analizuje trendy,  
5. automatyzacja wysyła alerty,  
6. manager podejmuje decyzje.

### **Workflow predykcyjny**
1. dane historyczne → model predykcyjny,  
2. model generuje prognozę,  
3. automatyzacja reaguje na prognozę,  
4. system tworzy zadania lub alerty.

---

## 7. Automatyzacje BI

Moduł BI współpracuje z silnikiem automatyzacji IF→THEN.

### **Przykłady automatyzacji**
- IF predykcja ryzyka > 70% → THEN utwórz zadanie „analiza ryzyka”,
- IF prognoza kosztów > budżet → THEN wyślij alert finansowy,
- IF kampania spada poniżej konwersji → THEN uruchom kampanię reaktywacyjną,
- IF projekt opóźniony → THEN aktualizuj harmonogram,
- IF przychody spadają → THEN wygeneruj raport sprzedażowy.
