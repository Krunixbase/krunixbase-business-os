# Krunixbase Business OS
### Modularny system operacyjny dla biznesu — CRM, Projekty, Budżety, KSeF, Płatności, BI, Automatyzacje, AI

Krunixbase Business OS to nowoczesny, modularny i skalowalny system operacyjny dla biznesu.  
Projekt łączy zarządzanie klientami, projektami, finansami, dokumentami, automatyzacjami oraz analityką w jeden spójny ekosystem.

System został zaprojektowany z myślą o firmach, software house’ach, integratorach, freelancerach oraz społeczności open‑source.

## 🧱 Architektura

System składa się z pięciu głównych warstw:

- **UI/UX** — nowoczesny interfejs, responsywność, panele boczne  
- **API** — REST, webhooki, autoryzacja tokenowa  
- **Logika biznesowa** — moduły operacyjne i premium  
- **Baza danych** — PostgreSQL, JSONB, indeksy, migracje  
- **DevOps** — CI/CD, kontenery, monitoring, alerting
