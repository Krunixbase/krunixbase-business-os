# Krunixbase Business OS

Kompletny system operacyjny dla firm:
CRM, kampanie marketingowe, automatyzacje IF→THEN, zadania, workflow, raporty, KSeF Business, integracje.

Repozytorium zawiera:
- historyczną ofertę współpracy,
- pitch deck systemu,
- prezentację systemu,
- dokumentację modułów,
- plan wdrożenia,
- plan rozwoju.

System jest gotowy do wdrożenia i może stać się centrum operacyjnym firmy.

## Historyczna oferta
Dokumenty w folderze `/docs/offer` przedstawiają pełną ofertę współpracy przygotowaną dla AJAI.
Oferta została udostępniona publicznie jako element portfolio i przykład procesu partnerskiego Krunixbase.
